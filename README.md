# keywordss-front


### build
一回だけbuildしたい

```
npm run build
```

### watch
動的にbuildかけたい

```
npm run watch
```

### start
サーバー起動(自動的にブラウザ開きます)
キャッシュしないようにしてるのでスーパーリロードする必要ないです
corsも許可するようになってる(はず)

```
npm start
```
